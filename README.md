TAHANAN

A Family Care & Connection App
Built for Filipino families, wherever they are

Core Features

1. Family Circle (Code System)
Each family has a unique code to securely connect members in one private space.

2. Care Check-ins
Daily or scheduled prompts like:

“Kumusta ka?”
Medicine reminders
Wellness check-ins for parents and elders

3. Emergency Connect
Quick-access family alert system for urgent situations with instant notifications and chat.

4. Family Schedule
Shared calendar for:

family dinners
birthdays
fiestas
important events
